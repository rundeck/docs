% JSON Data Converter
% Greg Schueler

Parses JSON text into a a Java object.

The [[page:manual/content-converters/html-table-view.md]] plugin can render 
this as a HTML Table.

## Input Types

* `application/json` - `String`

## Output Types

* `applicaiton/x-java-map-or-list` - `Map` or `List`
